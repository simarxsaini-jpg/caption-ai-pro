Adobe Premiere Pro — Import Guide
Follow these steps to add your captions to your timeline
1
Download the .SRT file using the "Adobe SRT" button above. Save it in the same folder as your video file.
2
Open Adobe Premiere Pro → Open your project → Import your video file into the Project panel.
3
Import the SRT file: Go to File → Import → Select your .srt file → Click Import. It appears in your Project panel.
4
Drag SRT to Timeline: Drag the .srt file from Project panel onto your timeline, directly above your video track. Premiere auto-syncs the captions.
5
Style your captions: Click any caption block → Open Essential Graphics panel → Change font, size, color, position. Select all captions and apply style globally.
6
Recommended fonts for Punjabi/Hindi: Use Noto Sans Gurmukhi for Punjabi, Mangal or Noto Sans Devanagari for Hindi. Set font size to 36–42px, white text with dark shadow.
7
Export with captions: Go to File → Export → Media → In Export Settings, check "Export Captions" → Choose Burn Captions Into Video (baked in) or Sidecar File.
